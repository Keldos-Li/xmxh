中美博弈的战略思路

德林社 发表于 2020-08-25 分类于 政治 阅读次数： Valine：

原作者: 尺度APP 德林社; 发表于 2019-01-29。

原编者(尺度APP)按：

中美经贸问题新一轮谈判正式开启，美国当地时间28日，中方代表团抵美，将与美国方面进行高级别磋商。
从去年2月算起，中美这场史无前例的贸易博弈，从春天到冬天，迄今已将近一年时间。
回顾这一年时间的发展脉络，中美双方的战略思路是什么？终局将会如何？
德林社最近上线了清华教授韩秀云的经济课程，课程第一模块主要讲中美贸易博弈，观点非常透彻，听完让人豁达。今天，在中美贸易磋商的关键节点，我们挑选了其中一课开放给大家。
除了中美贸易问题，如果你想了解股市、楼市、汇市等热点经济问题，欢迎订阅韩秀云教授的经济课，系统掌握经济学的底层思维和能力。

美国采取的是极限施压的战术，中国采取的是底线思维的策略。

让我们先来看看美国的极限施压的战术。

一、美国的极限施压战术

特朗普有一本自传叫《交易的艺术》，在这本书中谈到，他做生意有四个阶段：

  1. 提出惊人目标
  2. 大肆宣传
  3. 决策反复摇摆
  4. 获得直观的结果

他这样写道：提出一个远高于预期的条件，让对手无从下手；反复无常的变化给对手施加压力；给出次优条件让对手急于接受了事；最后达到最初想要的结果。

大家看，特朗普就是用这些招数来对付中国的：

  1. 从今年的4月份开始，特朗普用三轮喊话，那是吓唬中国；
  2. 从5月开始，美国用三轮谈判，狮子大开口，让中国无所适从；
  3. 从7月开打，他先打500亿，后打2000亿美元，想把中国打怕；
  4. 从12月1日，又提出90天的中场休息，高挂休战牌。

让我们举例说明，特朗普是怎么对中国政府极限施压的呢？

在2018年11月2日，美国总统特朗普和中国领导人通了电话，他表示已经让自己手下正在起草报告。他给市场的感觉像真的一样。市场是一片欢呼，美国股市上涨4%，中国股市上涨2%。

可是等他的中期选举完成后，他马上就变脸了。特朗普在11月27日说，他预计会推进关税上调的举措，就是把价值2000亿美元的中国商品的关税从10%上调至25%，他表示不可能接受北京方面，推迟提高关税的要求。

他为什么这样说？因为四天后，特朗普要与中国领导人举行峰会了。特朗普在接受《华尔街日报》采访时表示，如果中美不能达成协议，他将会对另外价值2670亿美元的中国商品加征为10%或25%的关税。

这是他惯用的伎俩。他用的又是极限施压战术。快要和中国领导人见面，他先放出风来，告诉说如果谈判不成功，他又将如何如何，这不明显是恐吓和极限施压吗？

他手里有牌吗？他这样做，美国企业答应吗？美国百姓能接受吗？他这是虚张声势。结果12月1日这天，在阿根廷的首都布宜诺斯艾利斯召开20国峰会，在两国元首晚宴上，他的脸又变了。

美国宣布，在2019年1月1日不再继续给中国的2000亿美元商品提高25%的关税，有3个月的时间坐下来谈判。美国挂起了休战牌，休战3个月，看谈判结果再决定打不打？这是吊中国的胃口，就是说，我看中国的表现再决定打不打你，都是我说了算。美国表面上要减少贸易赤字，多进口商品占领中国的市场，来解决美国经济的发展问题，但它最终目的只有一个，那就是遏制中国高科技的发展，不想让中国赶上美国，不想让中国和美国展开竞争。

大家看，这可是你美国提出的90天休战，不是中国提出来的，既然你美国这么牛，这么不可一世，你为什么要高挂休战牌呢？是你抗不住了吗？说对了，就是美国扛不住了，所以他要高挂休战牌，90天谈判。

二、中国政府的底线思维

在这场贸易战中，中国政府是怎么考虑的呢？中国政府是底线思维的策略。

美国和我们开打了，结果最差能差到哪里去？如果最差的结果我们都想到了，并且能扛得住，我们就可以跟美国博弈下去，最后看谁先扛不住？

中国是以不变应万变。不管你如何着急攻城，我稳坐钓鱼台。看着你出招，总有你出招尽了的时候吧。我最差又能差到哪里去？再有，你美国为什么不找别的国家的茬呢？因为其他国家根本不值得你打。你跟我打，说明了我的实力不弱啊！

敢于应战这是中国政府有勇气的地方。初中时我学过一篇古文叫《曹刿论战》，曹刿说，打战要：一鼓作气，再而衰、三而竭，彼竭我盈，故克之。

美国和中国打贸易战，先打了500亿美元，这是一鼓作气，又打了2000亿美元，这是再而衰，如果再打下去，那就是三而竭。

所以打到第二步时，美国说，停！咱们停下3个月，不再打了，咱们先谈谈后再做决定。如果接着打下去，美国可能就会三而竭了。

三、中国面对这种情况怎么出牌？

我们需要谋略。就是如何对付美国出的招，破了他的招。

我们要发展高科技产业，不能光靠企业自己单枪匹马，单打独斗，中国政府需要做好准备，为高科技企业的保驾护航。如果我们全部答应美国的条件，那就会前功尽弃，让中国经济冷上加冷，彻底打击我国的出口企业。

如果我们豁出去不怕死，那美国有什么招呢？他最想看到中国政府软下来，跟他求饶，答应他所有条件，签订城下之盟。

这对中国政府是一个考验，怎样面对呢？我们即不能让美国的阴谋得逞，同时又能达到和美国的斗而不破？

美国早就算计好了给中国商品加税，中国企业有多么倒霉，但是他们没有算到的是，加税后美国经济的受损程度有多大？他的这种强硬态度结果如何呢？那就是中美两败俱伤，没有赢家。打到这时，他发现这样下去，美国吃不消。

于是，美国高挂休战牌90天。给了两国缓冲和思考的时间。

我们问，当今美国可以离开中国而独善其身吗？不可以，他只是想拿到自己的最大利益，又不肯放掉中国市场。他是不情愿让中国长大，又离不开中国市场，这就是美国现在矛盾的心理状态。所以，他们提出再谈判。谈就谈呗，谁怕谁？

特朗普显然把这次中美经贸博弈当成了一笔“大生意”。目前的态势，有点像赌徒赌博一样。他想迫使中方心里发毛，最后在谈判桌上作出让步？

特朗普为了赢下这一局，他派来老奸巨猾的美国贸易代表莱特希泽带队来和中国进行贸易谈判。我在前面曾讲过此人，这里有必要再介绍一下莱特希泽，让大家了解这个人。看看我们后面的这场硬仗怎么打？

莱特希泽现年已71岁了，他是一员老将，他开始是干律师出身的，不久就步入了政坛。网上是这样介绍他的，说莱特希泽生于冷战，成名于80年代的日美贸易战，这次再度受命于中美贸易战。他是一个谈判老手，非常不好对付。我在这里给大家讲他的两个故事，看看此人是多么老辣、多么诡计多端。

1981年，年仅34岁的他获得委任，去和苏联就解除谷物禁运进行谈判，当时苏方代表送给他，一大罐古巴的雪茄，这里有两层含义：

表面上看，这是先礼后兵。其实却暗示1962年古巴的导弹危机。言外之意，一旦咱俩谈不拢，苏联方面不排除重启冷战。不仅如此，当时苏方认为他这么年轻，一定抽不了这么呛口的雪茄，就想看他当场出丑。

没想到，莱特希泽是一言不发，打开雪茄罐，一根一根抽了起来，最后抽到一根不剩，把整个没有窗户的会议室弄得是乌烟瘴气，苏方代表是面面相觑。结果是弄巧成拙。

在谈判中莱特希泽的姿态，非常强硬，在谈判面临破裂的最后关头，他索性抽身离场，还发出一封公函向苏联政府表明，谈判终止。结果迫使苏方妥协让步，最终达成了有利于美国的协议。

凭着这次出色表现，莱特希泽两年后，获里根总统任命为副贸易代表，负责打另一场硬仗，就跟当时经济如日中天的日本进行经贸谈判。

1985年，希特莱泽接到任务，与日本就钢铁和汽车进口问题进行谈判。在谈判桌上发生了戏剧化的一幕：莱特希泽根本不想看日本提出的方案，他故意把日方给他的谈判清单，折成纸飞机扔向对方，让日本谈判人员完全没想到，这令他们方寸大乱。

紧接着当日方贸易代表发言时，他又心不在焉地拆开面前的麦克风零部件在把玩。他还时不时讲些粗俗的笑话，使对手放松警惕，以诱导对手自己陷入自己的设置的“圈套”中。

谈判的结局是，日本被迫降低汽车等行业关税。这一战让莱特希泽是大获全胜。我们看，他在1985年对付日本使用的就是“301调查”。

从2017年起，美国对中国也发起的“301调查”，幕后就是由他主导的。他找到了《1974年贸易法》中“301条款”，并把这一条款利用到中国。301条款是允许美国对中国被指，在知识产权和技术转让方面，存在的不公平贸易行为，展开调查。莱特希泽认为，中国对美国构成了生死存亡的威胁，他的焦点就是试图打乱中国在高科技领域中的崛起。

面对这样的谈判对手，我们怎么应对？中方的谈判团队那是昼夜奋战，正在制定反制方案，怎样来克制他这样的谈判对手。

我们说，中美贸易战最后比的是什么？就是比看谁能扛得住？看谁扛不住？我的观点是，中国扛的住，美国扛不住。

中国有大量农业人口，这是贸易战的天然屏障，美国农民都得去市场购物，物价高了他们无法承受。还有中国的城市人口，拿上几百元到农村背上几袋棒子，就可以扛过几个月，你美国人扛的住吗？消费价格猛涨了，美国人会上街抗议的。

今天特朗普使出的这些极限施压、他一面发狠、一面装笑，声东击西，出尔反尔，他以为会让我们失了分寸，慌了手脚。他想错了，无论他怎么表演，我们知道他万变不离其中，就是想方设法遏制中国发展，特别是不让中国高科技产业发展。

他的司马昭之心，那是路人皆知，我们当然要布防，我们当然要提高警惕。我国政府用的是底线思维策略，来对付特朗普的极限施压，逼着对方又坐到了谈判桌上。
